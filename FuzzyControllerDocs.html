JS Fuzzy controller engine

The fuzzy engine takes in a list of fuzzy variables F and if-then rules of the form {y,L} where y is the output value and L is the list of values that the F variables can take.

For example:

{Y:"HIGH",F:["HIGH","","LOW"]}

equals the IF-THEN rule:

if F(1) = 'High' AND F(3) = 'LOW' THEN Y = 'HIGH'

A fuzzy variable itself is a list of fuzzy values (sets) that span some range of crips values.

The output of the engine is a fuzzy variable Y. The resulting Y value can be some weighed combination of the sets in Y so to get a crisp value the output value Y must be defuzzified. The defuzzification function is not part of the engine as multiple implementation exist for such a function.

Given the defuzzification function DF() the crisp y output of Y is

y = DF(Y)
